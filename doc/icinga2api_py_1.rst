icinga2api\_py layer 1
=======================

This selection of modules builds the first (bottom) "layer" of this library. See :ref:`doc-structure` about layers.
The reference is structure from bottom to top (top building on bottom). It starts to get interesting at
:ref:`ref-api-module`.

The first layer is mainly about constructing a request for the Icinga API in an easy way, also providing a solid basis
for the other layers to extend on. Classes here often extend classes of the `requests` library.
